#### Web development

All the languages below have been worked either during class or during my free time on sites such as [Coursera](https://coursera.org) or [codefinity](https://codefinity.com) or [mimo](https://mimo.org).

![Static Badge](https://img.shields.io/badge/html-FF8A00) I learnt HTML during highschool and kept improving my skills in this language.

![Static Badge](https://img.shields.io/badge/css-4269B5) CSS is a simple language, i mean when you know how to use something you are generally done learning this thing as long as you can remember it.
The only that i don't like about CSS is that it's more front-end than back-end and i prefer back-end over front-end.
I recently learnt how to use framework such as bootstrap or tailwind CSS to use it more efficiently.

![Static Badge](https://img.shields.io/badge/javascript-169BCF) Javascript is one of my favourite languages. I'm using it with Vanilla JS, React.js, Vue.js and will soon start Node.js

I don't really like it but it is very practical to understand the concept of memory inside of a computer.
Plus it hasn't the OOP included like C++, JAVA or Python. That makes it a language less important to me.

![Static Badge](https://img.shields.io/badge/java-EC2025) Java was taken as an example to learn the OOP.
I really like the way i can program classes in java but there is still one thing Java lacks.
When you have a simple task to do, you should not use OOP rightaway. 
For example, fonctional programming can be much more efficient i think for precise tasks.
But, OOP is very strong because of its maintainability and tha fact that it's reusable in most of its parts.
